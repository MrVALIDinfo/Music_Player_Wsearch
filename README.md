# Music_Player_Wsearch

A simple web-based music player with search functionality. You can search for songs, view album covers, and play/pause tracks. Designed with a clean, Spotify-inspired UI.

## Features

- Search for songs, artists, and albums
- Play and pause tracks
- Skip to next or previous tracks
- Progress bar with Spotify-like style
- Responsive design for desktop and mobile
- Stylish background video and semi-transparent player interface

## Technologies Used

- HTML5
- CSS3 (Flexbox, responsive design)
- JavaScript (Vanilla JS)
- MusicBrainz API for searching music
